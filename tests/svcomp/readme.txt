Some selected examples taken from the Software Verification benchmark, used in SV Competitions. See:

   https://sv-comp.sosy-lab.org/2021/benchmarks.php

The full benchmark consists of giga-bytes of "verification-tasks" :| Since we don't have a translator C or Java to OOX yet, this folder includes some of them, taken from the following category:

* Selected verification-tasks from the "JavaOverall" category. Generally I choose those tasks that seem to be challenging for participants of 2021 SV Competition.


For each verification task, I include the original task and target program, for reference.
